Carbon

Lightweight, mobile-first code editor for HTML/CSS/JS. Runs entirely in your browser with a floating keyboard for coding on the go.

Open Editor View on GitHub

Why Carbon?

Optimized for Mobile

Custom floating keyboard overlay adds braces, tabs, arrows, and other programming keys so you can code efficiently on a phone.

No Installs

Open in your browser, write code, and preview instantly. No accounts, saves locally, nothing to install.

Quick Prototyping

Perfect for prototyping and quick sketches — supports HTML, CSS, and JS in one file.

Screenshots

How It Works

  1. Open Carbon in your mobile browser.
  2. Write HTML, CSS, or JS in the responsive editor.
  3. Use the floating overlay for fast typing and cursor control.
  4. Preview your work instantly in the same tab.
  5. Saves your work in localStorage.
  6. Install as PWA for app-like experience.
